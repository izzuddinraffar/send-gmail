# send-gmail

## Project setup
Step 1: 
```
git clone https://github.com/izzuddinraffar/send-gmail.git
```

Step 2: 
```
py -3 -m venv <yourfolderapp>
```

Step 3: 
```
<yourfolderapp>\Scripts\activate 
```

Step 4: 
```
pip install Flask
```

Step 5: 
```
pip install -r requirements.txt
```
